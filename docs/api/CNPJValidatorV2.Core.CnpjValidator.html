Fornece métodos utilitários para sanitização, validação, cálculo de dígitos verificadores e formatação de números de CNPJ (Cadastro Nacional da Pessoa Jurídica), com ou sem formatação.

Suporta CNPJs alfanuméricos contendo letras maiúsculas e dígitos, conforme regras da Receita Federal.
